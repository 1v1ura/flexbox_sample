Flexboxの基本型

Flexboxレイアウトの構造は、「Flexコンテナー」と呼ばれる親要素にの中に、「Flexアイテム」と呼ばれる子要素を入れるだけです。 親要素にdisplay:flex;を指定するとその要素は「Flexコンテナー」となり、自動的にその子供要素は「Flexアイテム」となります。

親要素にdisplay:flex;を指定するだけで横並びレイアウトを簡単に作れます。

「Flexアイテム」が横並びになるのは、子要素の配置方法を指定するflex-directionプロパティの初期値にrowが設定されている為、下記のような挙動になります。

Item 3
Item 4
.flex-container {
  display: flex;
}

もし親要素がインライン要素の場合はdisplay: inline-flex;を指定します。

flex-grow

「Flexコンテナー」に余白があった場合に、flex-growの指定がある「Flexアイテム」を他の「Flexアイテム」と比較してどのくらいの比率で伸ばすかを指定

flex-grow: number;
Item 1
Item 2
Item 3

  display: flex;

.flex-container .flex-item:first-child {
  flex-grow: 1;
}
.flex-container .flex-item:last-child {
  flex-grow: 2;
}
